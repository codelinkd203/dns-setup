# DNS Home Setup Instructions
Hi, this is a tutorial on how to setup the DNS on your home network. Simply follow the steps below to set it up.

## Step-by-step Guide
1. Connect with admin on Google Chat (email is available [here](mailto:iammrbeastbackup@gmail.com))
2. Set up the DNS IP address (availible if owner or third-party is contacted, owner will give for free) in DNS settings **on home network**.
3. The admin will tell you to type in a link on your iPad *while you are connected*. The website will ***not*** load.
4. Wait a couple of seconds, and then the admin will activate your home router into the whitelist.
5. You may now use the DNS at home!
## Requirements
You must have something to bribe an admin with, such as candy or money. The admin puts time into allowing you in.
## Security
The DNS **does keep logs**, but it is **anonymous** and secure. Your **router IP address** will be stored, but it **will not be linked to you as a user**. Only admins can access these logs and IPs.

Any override may be turned of **at any time**, by **any admin**. You may request reactivation via Google Chat.
