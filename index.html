DNS Home Setup Instructions

Hi, this is a tutorial on how to set up DNS on your home network. Simply follow the steps below.

Step-by-step Guide

  1. Connect with admin on Google Chat (email available here)
  2. Set the DNS IP address in your router settings (get the IP here)
  3. Open the provided link on your iPad while connected (it will not load)
  4. Wait a few seconds for whitelist activation
  5. You may now use the DNS at home

Requirements

Access approval is required. Admins manage access manually.

Be respectful when requesting access.

Security

The DNS keeps logs, but they are anonymous. Your router IP address may be stored, but it is not directly linked to you as a user.

Access may be revoked at any time by an admin.

You can request reactivation via Google Chat.
